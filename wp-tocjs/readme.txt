It is a WordPress plugin that makes Table of Contents automatically.

== Description ==

Beautiful styles are provided by default, so you can use it without complicated editing and setting.

Recommended for those who want to easily create a simple Table of Contents.

[Plugin page](http://e-joint.jp/works/wp-tocjs/)

自動的に記事の目次を生成するWordPressプラグインです｡

シンプルな目次を手軽に作成したい方におすすめです。

[プラグインページ](http://e-joint.jp/works/wp-tocjs/)

[Github](https://github.com/ejointjp/wp-tocjs)

== Installation ==

= Install =

1. Get "WP TocJS" from "Plugins > New add" on the WordPress dashboard.
2. Activate this.
3. Set it on the "Settings > WP TocJS" in the dashboard.
4. You can set TOC show/hide individually on the post/page edit screen.

= インストール =

1. 「プラグイン > 新規追加」から「WP TocJS」を入手します。
2. プラグインを有効化します｡
3. ダッシュボードの「設定 > WP TocJS」で初期設定をします。
4. 投稿/固定ページの編集画面で、目次の表示/非表示を個別に設定できます。
